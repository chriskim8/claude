Pattern behavior

On-demand evaluation — pattern assignments are resolved at render/export time. Edit the pattern and all residents update without re-applying.

Override tracking — manual changes to a pattern-generated assignment are flagged as overrides, so deviations are visible at a glance.

Cancel rule integration — if a post-call cancel clinic rule is active, it fires automatically on call nights generated by this pattern.
